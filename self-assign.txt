If we find someone that doesn't have both will be warned.

:CB: :	Programmer

:python: :	Python
:java: :	Java
:cplus: :	C++

:lua: :	Lua
:rust: :	Rust
:php: :	PHP
:golang: :	Golang
:shell~1: :	Shell Script (or any other implementation of it eg. Bash)
:powershell: :	Batch Script
