[Languages Roles]

React with following emojis to get a certain role. (also look below)
**First** get the Programmer role and optionally pick one or more language roles afterwards.
Without the Programmer role, you cannot get a language role.

:CB: :	Programmer

:python: :	Python
:java: :	Java
:cplus: :	C++
:js: :	JavaScript
:csharp: :	C#
:c_: :	C
:lua: :	Lua
:rust: :	Rust
:php: :	PHP
:golang: :	Golang
:shell~1: :	Shell Script (or any other implementation of it eg. Bash)
:powershell: :	Batch Script
